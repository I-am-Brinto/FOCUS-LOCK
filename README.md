# Focus Lock

Focus Lock is a Chrome extension designed to help users stay focused during Pomodoro sessions.

It combines a distraction-blocking system, daily focus tracking, a lightweight to-do list, and simple gamification features in one clean interface.

## Features

### Pomodoro Timer
- Default focus session: 25 minutes
- Default break session: 5 minutes
- Focus and break durations can be changed from Settings
- Timer continues through the extension background system even when the popup is closed

### 15-Second Setup Window
Before every focus session begins, Focus Lock gives you 15 seconds to open all websites and tabs you need.

After the setup time ends, opening new websites is restricted during the focus session.

### Website Lock System
During focus mode:
- Previously opened websites remain accessible
- New websites are blocked
- A blocked page asks whether the website is genuinely necessary
- Urgent access becomes available after a 1-minute waiting period

This delay is intentionally designed to reduce impulsive browsing.

### Daily Focus Target
Set a daily focus goal in minutes.

The extension tracks how much focused time you complete each day and displays progress through a progress bar.

### To-Do List
Focus Lock includes a compact to-do list inside the popup.

You can:
- Add tasks
- Mark tasks as completed
- Delete tasks
- Clear completed tasks

Tasks are stored locally, so they remain available after closing Chrome or reopening the extension.

### Gamification System
Focus Lock rewards consistency.

- Complete your daily focus target to earn one Golden Coin
- Complete targets on consecutive days to build a streak
- Coins and streak data are stored locally and do not reset when the popup closes

### Language Toggle
The popup supports:
- English
- বাংলা

English is the default language. Your selected language is saved locally.
