# How to Install and Use Focus Lock

## 1. Download the Extension

1. Open this GitHub repository.
2. Click the green **Code** button.
3. Click **Download ZIP**.
4. Save the ZIP file on your computer.

## 2. Extract the ZIP File

1. Find the downloaded ZIP file.
2. Double-click it to extract it.
3. A folder will be created after extraction.

Do not delete or move this folder after loading the extension in Chrome.

## 3. Open Chrome Extensions

1. Open Google Chrome.
2. Type the following address in the Chrome address bar:

```text
chrome://extensions
